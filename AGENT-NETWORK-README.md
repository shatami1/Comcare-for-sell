## 🎯 How The System Works

---
<div style="background:#1a237e;color:#fff;padding:12px 24px;border-radius:18px;margin-bottom:16px;">How The System Works</div>
---

### For Agents
1. **Apply** → Fill out application on main site
2. **Get Approved** → Admin approves within 24 hours
3. **Receive Branded Page** → Get custom URL like: comcare.store/agent/yourname
4. **Share Link** → Promote on social media, email, business cards
5. **Earn Commissions** → 25% on every order through their link

### For You (Owner)
- **Admin Portal** shows ALL business data
  - Total: 47 orders, $18,500 revenue
  - You get 75% ($13,875), agents get 25% ($4,625)
- Track all agents, orders, customers, inventory
- See which agent brought which customer
- Manage entire network from one dashboard

### For Customers
- Find agent through:
  - Social media posts
  - Google search for "ComfortCare [city]"
  - Agent business cards
  - Referrals
- Order through agent's branded page
- Transaction goes through YOUR main system
- Agent gets credited automatically
